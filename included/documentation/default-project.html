Default Project

Splash

When you first open Code Rocket you will be greeted with a demo Rocket, an animated splash screen. This features an animated SVG rocket, welcoming text, and a background image.

Features

By clicking the down pointing arrow on the Splash page you be be animated to the Features page which lays out some of Code Rockets core features.

Shortcuts

On the final page of the Code Rocket Default Project there are some keyboard shortcuts for the most important editor functions, these should be memorized for complete Rocket control.
